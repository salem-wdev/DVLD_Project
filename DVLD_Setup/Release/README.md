# DVLD Installation Guide

## Requirements
- Windows 10 or Windows 11
- Administrator privileges
- The installation package must remain intact. Do not remove or rename any files.

## Installation

### Step 1: Install SQL Server LocalDB

Locate the SQL Server LocalDB installer included in this folder and run it.

- If LocalDB is already installed on your computer, you can skip this step.
- If it is not installed, complete the installation before continuing.

### Step 2: Install DVLD

Run:

setup.exe

Then follow the installation wizard:

- Next
- Next
- Install
- Finish

### Step 3: Launch DVLD

Start DVLD from the Start Menu or the desktop shortcut (if created).

## Troubleshooting

If the application cannot connect to the database:

- Make sure SQL Server LocalDB is installed.
- Restart the computer if LocalDB was installed during this session.
- Reinstall DVLD if necessary.
